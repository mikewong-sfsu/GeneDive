Disambiguation Manager

Used to manage data involving DGRs with many symbol IDs.

  • Uses GeneDiveAPI.js to provide additional info about DGRs that resolve to multiple IDs.
  • The search.js file calls disambiguation.js's resolveIds function in the event that a typeahead element is select by a user, and the associated gene requires disambiguation.
