Highlight

This component handles the highlighting of rows within the table-view. In GeneDive, highlighting refers to applying a color filter of rows that meet a user-specified criteria.

  • highlight.js allows for the creation of a Highlight object, which controls the state of a highlight variable to apply to an interaction's Regex if a match for a highlight term is selected by the user.
  • controller.js instantiates a Highlight object and renders a highlighted row based on the data maintained by that object.
