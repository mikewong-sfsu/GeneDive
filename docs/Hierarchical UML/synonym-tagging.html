Synonym Tagging

In some cases, certain DGD IDs are variations of a root ID, or are otherwise so similarly named to a search term that the user might like to see them in a search result when he did not specify them. This component of GeneDive is responsible for identifying these tags.

  • synonym.js enables the creation of Synonym classes, which map IDs to related or similarly named tags.
  • Controller.js creates a Synonym class via synonym.js, and calls its findSynonyms() method to identify tags associated with the search term.
