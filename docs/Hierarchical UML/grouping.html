Grouping

Grouping refers to the ways in which search results are ordered in the table view. Grouping order is determined by a sort on a key provided by grouper.js.

  • grouper.js enables the instantiation of a Grouper object, which groups interactions based on a key that is specified in its group() method.
  • controller.js instantiates a Grouper object. The Grouper can be provided with a list of interactions as input, and the output is the grouping order.
