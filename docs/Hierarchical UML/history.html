History

This component keeps track of a user's history of interactions with GeneDive. A record of activities is kept on a stack that can be pushed to and popped from at will. Users can import and export their histories.

  • history.js enables the creation of a History object, which handles the recording and navigation of the various states that a user puts the GeneDive application into.
  • controller.js creates a History object from history.js, and forwards information about user input to this object.
