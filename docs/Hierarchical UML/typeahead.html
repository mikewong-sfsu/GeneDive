Typeahead

This component dynamically provides suggestions to the user based on the input that they provide to the search bar.

  • Powered by Twitter's typeahead javascript bundle (included in project).
  • Calls to the typeahead Bloodhound component are made via search.js's initTypeahead() function.
  • An instance of search.js's Search class is created in the controller class, and the Search object's functions are accessed when users provide input (handled by controller.js)
