Filter

The filter component provides users with the ability to constrain results based on minimum confidence score.

  • Determined by slider bar value, which is carefully tracked via probabilityfilter.js.
  • Button clicks can automatically set the confidence score and corresponding slider to a hard-coded value, also handled via probabilityfilter.js.
  • Controller.js creates an instance of probabilityfilter.js, and monitors that instance's properties in order to update the user's desired minimum probability.
