View Manager

Handles rendering of data to both the table and graph views.

  • controller.js instantiates instances of objects, each of which process data based on user-provided input and return information that the controller uses to correctly render to both the table and graph views.
  • controller.js contains several functions which are called when the user provides input, and which return data after forwarding details about siad user input to corresponding instantiated object methods.
