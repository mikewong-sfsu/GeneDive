Detailed Description

File auto importer.

Author
Jack Cole jcole.nosp@m.2@ma.nosp@m.il.sf.nosp@m.su.e.nosp@m.du
Date
2018-01-08

This script automatically imports all javascript files in it's directory. It will return a dictionary with the file names (without their extensions) as the index, and the imports as values. Files can be ignored by using the IGNORED_NAMES constant.
