Highlight Highlight::highlight (   interactions)
inline

Highlights interactions that match terms.

Searches every interaction's context and the two DGRs for the term with Regex, and changes it's highlight variable to true if it's a match, and false if not. The minimum length is 2, so no one letter searches.

Parameters
interactionsThe interactions from the GeneDive search
