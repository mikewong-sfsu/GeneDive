Detailed Description

History (undo/redo) management API.

This class will store and load state history of GeneDive

History History::saveCurrentState ( )
inline

Saves the state of the GeneDive program.

This will take the current state of the GeneDive program, and convert it to a an object. The state is based off the the variables that describe the Table, the Graph, the Filters, Search, and more.

Returns
object The state of the GeneDive program

◆ saveCurrentStateToHistory()

History History::saveCurrentStateToHistory ( )
inline

Adds the current state to History.

This will add a new item to the state history, while removing any stats ahead of the current state If the view is currently loading, dictated by the spinneractive variable, then the state will not be saved.
