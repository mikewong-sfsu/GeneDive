Search Modes

1-Hop

Provide one Disease, Gene, or Drug entitiy (DGD) to view all direct interactions with that DGD. Provide multiple DGDs or DGD sets to view interactions between all input DGDs.

2-Hop

Provide two Disease, Gene, or Drug entities (DGDs) as input. GeneDive will show interactions between the two DGDs through up to one intermediary DGD.

3-Hop

Provide two Disease, Gene, or Drug entities (DGDs) as input. GeneDive will show interactions between the two DGDs through up to two intermediary DGDs.

Clique

Provide one Disease, Gene, or Drug entity (DGD) as input. GeneDive will show those DGDs that interact with the input DGD and one of its interactants.

Text Filtering and Metacharacters

Text filtering can be applied to search terms by formatting search queries with certain metacharacters. Click here to learn about common text filtering patterns and metacharacters. Some metacharacter usages are described below.
. Matches any single character except the newline character.
For example, /.n/ matches 'an' and 'on' in "nay, an apple is on the tree", but not 'nay'.
* Matches the preceding expression 0 or more times.
For example, ab*c matches "ac", "abc", "abbbc", etc.
.* Matches the preceding character, followed by zero or more other characters.
For example, s.* matches "s" and "saw" and "seed".

Using the Graph

Navigate

Click and drag on whitespace to move around.

Manipulate

Click and drag genes to move them around. Genes will stay where you leave them.

Shift-Click Gene to Add

Shift-Click on genes to add them to the search. Releasing shift will start a new search.

Ctrl-Click Gene to Start New Search

Ctrl-Click on a gene to start a new search with only that gene.
